# Dogfoodma / AFO Mobile Visual Runtime v0.03

## Purpose

v0.03 upgrades Dogfoodma from a basic Three.js runtime into a premium frontend MCP-style scene compiler.

The core principle is that agents should author high-level design and scene specs, not unstable raw Three.js strings.

## New v0.03 Layers

1. Premium design schema
2. Scene synthesis schema
3. Theme, HUD, material, lighting, layout, and control presets
4. Three.js object graph export
5. React Three Fiber component-tree export contract
6. glTF manifest export contract
7. Mobile performance budget validator
8. Premium mobile cockpit shell

## Recommended Agent Flow

1. Read `/llms.txt`.
2. Read `/mcp/tools` and `/mcp/schema`.
3. Compile design tokens with `/api/design/compile`.
4. Validate and optimize scene spec.
5. Export to one or more targets.
6. Compile mobile Worker page.
7. Generate receipt.
