# AFO Mobile Visual Runtime v0.01 Spec

## Runtime role

AFO Mobile Visual Runtime is the visual/frontend renderer layer for the AFO app factory.

It accepts a small scene spec and returns a mobile-first Three.js web app that can be served from a Cloudflare Worker.

## Non-goals for v0.01

- no full Figma clone
- no arbitrary WebGL authoring
- no desktop-first editor
- no heavy build pipeline
- no mandatory npm frontend framework

## Design constraints

- phone-first
- one canvas
- DOM HUD for text
- WebGL scene for motion and spatial UI
- CDN Three.js allowed
- Cloudflare Worker compatible
- backend binding vocabulary for D1/KV/R2
- scene spec before raw code

## Compiler phases

1. normalize scene spec
2. apply mobile budget
3. convert records to visual objects
4. generate HTML/CSS/JS
5. expose runtime routes
6. return receipt
