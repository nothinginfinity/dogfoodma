# Dogfoodma v0.04.1 — Template Extraction

## Purpose

v0.04.1 introduces the first reusable frontend template: `templates/premium-cockpit-app`.

The goal is to make Dogfoodma usable as a frontend layer for future app-factory builds without editing the runtime directly.

## Template Files

- `templates/premium-cockpit-app/README.md`
- `templates/premium-cockpit-app/scene.json`
- `templates/premium-cockpit-app/theme.json`
- `templates/premium-cockpit-app/bindings.json`
- `templates/premium-cockpit-app/template.manifest.json`

## Why This Matters

Dogfoodma is now two things:

1. A deployed runtime that renders and compiles frontend scenes.
2. A reusable frontend template system that other agents/chats can clone, theme, bind, and deploy.

## Next Target

v0.05 should integrate this template with Sandbox Builder, GitHub MCP, and Cloudflare MCP so a new app can be generated from the template and deployed as its own Worker.
